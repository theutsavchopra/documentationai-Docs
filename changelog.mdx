## Overview

Bragly releases updates regularly to enhance your documentation experience. This change log tracks all major and minor releases, highlighting new features, improvements, bug fixes, and breaking changes. Review it to understand what's new and plan your upgrades.

Stay informed by subscribing to our release notifications or checking this page periodically.

<Callout kind="info" collapsed="false">
  We're constantly improving Bragly. While smaller fixes may not always appear here, this page highlights the most important updates.
</Callout>

## Recent Releases

<Update label="May 2026" description="v2.1" tags={["feature","improvement","bugfix"]}>
  ## New Features

  - Email Campaigns let you send personalized review-request emails, import recipients, duplicate campaigns, retry failed sends, and honor unsubscribe preferences.

  - Thank-you email tools help you follow up from collection forms and review profiles after someone shares feedback.

  - Wall of Love Discover theme adds search, filters, and sorting so visitors can browse testimonials faster.

  - Testimonial widgets now include company logo displays, video highlights, quote lists, ribbon carousel, avatar ribbon, and avatar mosaic layouts.

  - Collection forms can generate QR codes for easier sharing.

  ## Updates

  - Unsubscribe and resubscribe handling now gives recipients clearer control over campaign emails.

  - Collection forms now support redirect messages and default tags for cleaner post-submit flows and easier review organization.

  - Imported testimonials now require explicit rating selection for clearer review data.

  - Review details now support sending manual thank-you emails.

  - Manual testimonial creation now includes rating selection and company logo uploads.

  - Brand Kit access now appears in plan details so teams can see availability before setup.

  ## Bug Fixes

  - Fixed the rating-required message so collection forms show clearer guidance when a rating is missing.

  - Fixed company-logo handling for form submissions.

  - Fixed widget layout fallback behavior.

  - Fixed Discover source filtering so selected sources return the expected testimonials.
</Update>
