## Overview

Bragly releases updates regularly to enhance your documentation experience. This changelog tracks all major and minor releases, highlighting new features, improvements, bug fixes, and breaking changes. Review it to understand what's new and plan your upgrades.

Stay informed by subscribing to our release notifications or checking this page periodically.

<Callout kind="info">
  Enable notifications in your Bragly dashboard to get email alerts for new releases.
</Callout>

## Recent Releases

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added AI-powered content suggestions to help you draft documentation faster.
- Introduced version pinning for docs pages, allowing you to lock content to specific releases.
- New embeddable search widget for integrating Bragly search into your app.

## Improvements

- Enhanced search relevance with semantic matching for technical terms.
- Reduced page load times by `>30%` through optimized asset delivery.
- Improved mobile responsiveness for all documentation themes.

## Bug Fixes

- Fixed markdown rendering issues with nested lists and tables.
- Resolved authentication token expiration during long editing sessions.
- Corrected export failures for large documentation sites.

</Update>

## Upgrading to the Latest Version

Follow these steps to upgrade your Bragly instance safely.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Export your documentation projects to avoid data loss.

    ```bash
    bragly export --project my-docs --format zip
    ```

  </Step>
  <Step title="Update Package" icon="download">

    <CodeGroup tabs="npm,yarn">
    ````bash
    npm update bragly@latest
    ````
    ````bash
    yarn upgrade bragly@latest
    ````
    </CodeGroup>

  </Step>
  <Step title="Restart Services" icon="refresh-cw">

    Restart your Bragly server and verify the version.

    ```bash
    bragly restart --check-version
    ```

  </Step>
  <Step title="Test Changes" icon="check-circle">

    Review the [upgrade notes](https://docs.bragly.com/quickstart#upgrading) and test key features.

  </Step>
</Steps>

<Callout kind="tip">
  After upgrading, clear your browser cache and regenerate static pages if using the static export feature.
</Callout>
