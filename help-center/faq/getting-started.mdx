## Create Your First Project

1. Log in to your Bragly account
2. Click "New Project" on the dashboard
3. Enter a project name and description
4. Click "Create"

Your new project will appear in your dashboard. You can now start adding docs and organizing them into folders.

```tip
Use short and clear project names for easy navigation.
```
