---
title: Cannot Create Project
description: Resolve project creation issues in Bragly.
---

## Symptoms

- "Create" button is disabled
- Error: "Project creation failed"

## Quick Fix

<Steps>
  <Step title="Check Project Name">
    Ensure the project name field is filled.
  </Step>
  <Step title="Refresh the Page">
    Sometimes session timeouts cause this. Reload and try again.
  </Step>
  <Step title="Contact Support">
    If the issue continues, email support@bragly.com with details.
  </Step>
</Steps>
